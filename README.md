# Digital_Thermometer

## Required Parts
Arduino UNO Board
Bread Board
Jumper wires
16x2 LCD Display
LM35 temperature sensor 
50k-ohm potentiometer

## The Build
|LCD Display | Arduino|
|--------|-------|
|VSS | GND|
|VDD | 5V|
|VO | Center Pin of Potentiometer|
|RS | Pin 12|
|RW | GND|
|Enable | Pin 11|
|D4 | Pin 5|
|D5 | Pin 4|
|D6 | Pin 3|
|D7 | Pin 2|
|A Bcl + | 5V|
|K Bcl - | GND|
